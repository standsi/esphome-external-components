# esphome-external-components

### This collection of modified and custom esphome components focus on low power functionalities using the ESP microcontrollers.  See the README file in each component directory for usage details.

* mlx90393 is a modified version of the standard magnetometer component with wake on change operations.
* ulp_flash is for esp32s2/s3 using fsm so **only supported through esphome 2025.12**
* lp_blink is for esp32c6 using lp core, supported on latest esphome
* risv_blink is for esp32s2 and esp32s3 risc-v ULPs, supported on latest esphome
